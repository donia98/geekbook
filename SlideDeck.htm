What is
JavaScript?
JavaScript makes
 web pages interactive
JavaScript basically does two things:
  • Make a webpage interactive aesthetically
  • Stuff happens when you hover over stuff and click on menus, things change color, etc.
  • Fetch data from the server without ever having to leave the page (e.g. Google search autocomplete)
  • This is known as AJAX  (Asynchronous JavaScript and XML)

JavaScript vs. jQuery
JavaScript & jQuery
  • Syntax is kind of clunky
  • Behavior isn't always consistent across different web browsers
  • People have attacked these problems by creating frameworks that wrap JavaScript's behavior nicely and make it so we don't have to worry about clunky syntax or cross-browser inconsistencies
  • jQuery is the most popular of these frameworks
  • jQuery has fantastic documentation:
  •  http://api.jquery.com/
